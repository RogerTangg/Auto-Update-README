# DevOps Auto-README Project

本專案示範如何使用 GitHub Actions 自動更新 README，並與 GitHub Projects Issue 連結。

---

## 專案目標
- 自動更新 README 的活動紀錄
- 與 GitHub Projects Issue/PR 連結
- 確保安全性與最小權限
